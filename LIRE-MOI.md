# ✂️ Salon d'Elysa — Guide de mise en ligne sur Vercel

Suis ces étapes dans l'ordre, c'est simple !

---

## Étape 1 — Créer un compte GitHub (gratuit)

GitHub est l'endroit où on va stocker le code de l'app.

1. Va sur **https://github.com**
2. Clique sur **"Sign up"**
3. Crée ton compte (email + mot de passe)
4. Confirme ton email

---

## Étape 2 — Mettre le projet sur GitHub

1. Une fois connecté sur GitHub, clique sur le **"+"** en haut à droite
2. Choisis **"New repository"**
3. Donne-lui le nom : `salon-elysa`
4. Laisse tout le reste par défaut, clique **"Create repository"**
5. Sur la page qui s'affiche, clique sur **"uploading an existing file"**
6. **Glisse-dépose tous les fichiers** du dossier `salon-elysa` que tu as téléchargé
   - ⚠️ Important : glisse aussi le dossier `src` avec son contenu
7. Clique **"Commit changes"**

---

## Étape 3 — Créer un compte Vercel (gratuit)

1. Va sur **https://vercel.com**
2. Clique **"Sign Up"**
3. Choisis **"Continue with GitHub"** — ça connecte directement les deux comptes

---

## Étape 4 — Déployer l'app sur Vercel

1. Une fois connecté sur Vercel, clique **"Add New Project"**
2. Tu verras ton repository `salon-elysa` dans la liste — clique **"Import"**
3. Vercel détecte automatiquement que c'est un projet Vite/React
4. Ne change rien, clique directement **"Deploy"**
5. Attends 1-2 minutes... 🎉

---

## Étape 5 — Ton app est en ligne !

Vercel te donne une adresse du type :
**`https://salon-elysa.vercel.app`**

C'est ton app ! Elle fonctionne sur téléphone et ordinateur.

---

## Étape 6 — Ajouter l'app sur l'écran d'accueil du téléphone

### Sur iPhone (Safari) :
1. Ouvre l'adresse dans **Safari**
2. Appuie sur l'icône **Partager** (carré avec flèche vers le haut)
3. Fais défiler et appuie sur **"Sur l'écran d'accueil"**
4. Appuie **"Ajouter"**
→ L'app apparaît comme une vraie appli sur ton téléphone !

### Sur Android (Chrome) :
1. Ouvre l'adresse dans **Chrome**
2. Appuie sur les **3 points** en haut à droite
3. Appuie sur **"Ajouter à l'écran d'accueil"**
4. Appuie **"Ajouter"**

---

## ⚠️ Important à savoir sur les données

Les rendez-vous sont sauvegardés **dans le navigateur du téléphone**.
- ✅ Elles restent même si tu fermes l'app
- ✅ Elles restent si tu redémarres le téléphone
- ❌ Elles sont perdues si tu effaces les données du navigateur
- ❌ Elles ne se synchronisent pas sur un autre téléphone

**Conseil :** utilise toujours le même navigateur sur le même téléphone.
